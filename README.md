
# Image Classifier (MobileNetV2)

A deep learning web app that classifies images using MobileNetV2 trained on ImageNet.

## Features
- Upload image
- Top-3 predictions
- Confidence scores
- Lightweight model for fast inference

## Tech Stack
- TensorFlow / Keras
- Flask
- HTML/CSS

## Deployment
Optimized for low-memory environments using MobileNetV2.
